# Blackjack Trainer

MBS（マリーナベイ・サンズ）公式ルールに対応した、ブラックジャックのベーシックストラテジーおよびカウンティングの特訓用Streamlitアプリケーションです。

## 🎮 搭載モード
- **通常モード（完全ランダム）**: 実戦形式でのストラテジー練習
- **特訓モード（難問限定）**: スプリットやソフトハンド等、迷いやすい手札を集中特訓
- **高速自動シミュレーション**: AIによる数万回の自動プレイと収支推移のグラフ検証
- **カウンティング特訓**: AIの自動プレイを観戦しながらランニングカウントを暗算する特訓
